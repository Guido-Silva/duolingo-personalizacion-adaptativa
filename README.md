# Sistema de Personalización Adaptativa — Caso Duolingo

**Curso:** Bases de Datos — Maestría en Ciencia de Datos e IA (UTEC)
**Año:** 2025-2026

## Descripción
Proyecto académico que propone un sistema de personalización profunda 
para la plataforma Duolingo, basado en detección de patrones de error 
específicos del usuario (gramática, sintaxis, género gramatical).

## Contenido
- **Parte 1:** Modelado Entidad-Relación (MER) de la plataforma Duolingo
- **Parte 2:** Diseño del sistema de recomendación adaptativa

## Tecnologías y conceptos aplicados
- Bases de Datos Relacionales
- Modelado MER
- Machine Learning (sistemas de recomendación)
- Diseño de flujos de trabajo orientados a datos# duolingo-personalizacion-adaptativa
